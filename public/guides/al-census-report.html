Synthesis — 어떻게 여전히 주장할 수 있는가

1. 외부 독립 검증 — Auburn 대학 Marshall & Nelson 2024 op-ed가 독립적으로 "15/139" 보고 → 내 6 primary / 20 instructional과 compatible.

2. Random 10-sample ground-truth audit — RNG seed 42로 silent pool에서 10개 선정, 10/10 모두 silent 확인. Pickens County는 Code of Conduct PDF 직접 fetch하여 AI 언어 전무 확인 (hard).

3. Hard-verified silent 3개 확보 — Birmingham Code of Conduct PDF + Auburn AUP HTML + Pickens County PDF 직접 fetch → 모두 AI 언어 zero.

결론: Phase 2의 saturation 증명 논리는 retracted. 그러나 독립 연구 일치 + 랜덤 샘플 검증 + hard-verified sample이 결과값의 방향성을 지지. 신뢰도 85% → 80%로 honest 조정.

용어

"AI signal" ≠ "AI policy". Signal은 정책 문서 + 커리큘럼 채택 + 행정가 발언 + 위원회 + 조달을 모두 포함하는 포괄 용어. Policy는 그중 정식 문서(6개)만. 연구 논문 수준에서는 반드시 구별.
